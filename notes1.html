Topic 1: The World Wide Web and HTML Basics

The World Wide Web
The web is made up largely of HTML (HyperText Mark Language) documents. The major components of the web include you, your computer, a web browser, the Internet and servers. HTTP (HyperText Transfer Protocol) is the protocol used when browsers such as 'Google Chrome, Safari, Mozilla Firefox etc.' make requests to servers.

  1. Computer
  2. Internet
  3. HTTP
  4. Servers

Basic HTML Elements
HTML elements are the basis for how a web browser displays a web page. They are usually made up of an opening and closing tags with contents in between. Below are some examples:

  1. <>content</b> - Makes content bold
  2. <em>content</em> - Makes content italic

* This video provides a good overview of what a HTML element is.

Some exceptions include 'img' and 'br' tags that do no have contents and are also known as void tags. Examples:

  1. <img src="url" alt="text"> - To display images
  2. <br> - For line breaking
* This video shows an example of a void tag.

HTML Attributes
HTML elements can have additional values known as attributes in their opening tags. Below is an example of the anchor tag that contain a attribute called 'href' which creates a link to a website:

  • <a href="url" alt="text">
* This video provides an explanation on HTML attributes and creating website links.

Inline/Block Elements
HTML elements can be either inline or block. The two behave differently, the span tag is used to apply style to groups of inline elements, while block elements create an invisible box around contents inside of them.
