Boxes Everywere
  • Everything on a webpage is a box
  • Boxes are defined with <div> tags; Placeholder contents are used to help us remember which boxes each of our tags refer to. For example, <div>image<div>
  • Class attributes helps remind us what are inside the <div> tag
Adding Style

There are 3 ways to include CSS styling on a web page:

  1. Write CSS in the <head> of your HTML
  2. Link HTML to a separate CSS file
  3. Write your style inline with your HTML

Positioning of Boxes

A <div> is a block element, hence, contents in different <div> tags appear one on top of the other. Thus, in order to position boxes side-by-side, the rule display: flex can be applied
